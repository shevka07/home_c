Перевод 12345678₁₆

Разбираем число по разрядам:

8 * 16⁰ = 8 * 1 = 8
7 * 16¹ = 7 * 16 = 112
6 * 16² = 6 * 256 = 1536
5 * 16³ = 5 * 4096 = 20480
4 * 16⁴ = 4 * 65536 = 262144
3 * 16⁵ = 3 * 1048576 = 3145728
2 * 16⁶ = 2 * 16777216 = 33554432
1 * 16⁷ = 1 * 268435456 = 268435456
Складываем полученные значения:

8 + 112 + 1536 + 20480 + 262144 + 3145728 + 33554432 + 268435456 = 305419496

Результат:

12345678₁₆ = 305419496₁₀

0 * 16¹ = 0 * 16 = 0
0 * 16² = 0 * 256 = 0

0 * 16⁴ = 0 * 65536 = 0
1 * 16⁵ = 1 * 1048576 = 1048576
